Welcome to DocIntel OS

This is a stateful multi-agent system orchestrating specialist nodes to process documents. Here is how it works:

  • Supervisor: Evaluates queries and coordinates specialist worker routing.
  • OCR Extraction Agent: Extracts text from raw PDF/image files.
  • RAG Retrieval Agent: Runs semantic index search over document text chunks.
  • Structured Output Agent: Parses text context into a strict Pydantic JSON structure.
